Widget f&uuml;r die Posts einer Facebook Fanpage (ohne Page Plugin per graph.facebook.com). Inklusive Filterung per Hashtag und vier schicken Styles. 

== Description ==
Datenschutzkonforme Einbindung der letzten Post einer Fanpage. Das Plugin zeigt den Stream einer Fanpage, ohne dass Daten an Facebook gesendet werden. Als Schnittstelle dient die Abfrage per graph.facebook.com. 

= Das k&ouml;nnt ihr einstellen: =
* Titel 
* Anzahl der Posts 
* Einschr&auml;nkung, dass nur Posts mit einem bestimmten Hashtag angezeigt werden
* Maximale L&auml;nge des Textes 

== Installation ==
1. Plugin herunterladen und aktivieren.
1. In den Widgets erscheint ein neues Widget mit dem Titel \"FuZo Facebookstream\".
1. Ziehe das Widget in eine deiner Widget-Areas.
1. Unter https://developers.facebook.com/apps kannst du oben rechts per Klick auf den gr&uuml;nen Button "Neue App hinzufügen" eine neue App erstellen
1. Speichere das Widget

== Frequently Asked Questions ==
= Wie kann ich die Farben anpassen? =
Die Farben lassen sich per CSS &uuml;berschreiben:
Beispiel: #facebook_widget.fuzo a {color:#efefef; background-color:#888;}

= Welche Einstellungen kann ich vornehmen? =

* Titel (optional)
* Anzahl der Posts 
* Facebook Seiten ID (Die Seiten ID findest du hier)
* Facebook App ID und Facebook Secret (So findest du diese
* Hashtag - optional
* Maximale L&auml;nge des Textes (optional)
* Auswahl aus vier vordefinierten CSS (K&ouml;nnen optional deaktiviert werden, so dass ihr eure eigenen Styles nehmen k&ouml;nnt)
